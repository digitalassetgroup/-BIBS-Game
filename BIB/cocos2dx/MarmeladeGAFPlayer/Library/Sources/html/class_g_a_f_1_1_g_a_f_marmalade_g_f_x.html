Detailed Description

Note
You must call onFrame() every frame after your rendering step is over or at the beginning of the frame.

To implemnet rendering playback other than Cocos2dx you must implement all functions listed in GAFMarmaladeGFX. For sample implementation you can check current Cocos2dx-bases implementation. Our advice is to contact gafmedia.com before you start to work on your own backend - most likely backend you need is already in development. We tryed to be as less demanding as we can in terms of backend features you need to implement GAFMarmaladeGFX. If your game engine features sprites, transformation and OpenGL ES shaders it should not take too long to implement playback. If you have only OpenGL ES immediate mode then things are more diffictult to you - you have to implement mentioned abstructions. The most important thing with immediate mode implementation is carefully set and restore OpenGL states.
