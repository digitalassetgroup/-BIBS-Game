Инструкция по эксплуатации.
1. Качаем кокос-айфон с http://www.cocos2d-iphone.org/download. Я проверял на 2.1rc, вы можете проверить на 2.0. Если ничего не будет компилироваться на 2.0 ничего страшного, с этим мы ничего делать не будем все равно.
2. Устанавливаем шаблоны проектов.
3. Созаем новый проект.
4. Удаляем исходные файлы сэмпла - их 6 AppDelegate, HellowWorldLayer, IntroLayer(.h + .m).
5. Добавляем вместо них содержимое папки cocos2d-iphone в исходники.
6. Добавляем содежимое папки Resources в ресурсы. При этом выбираем опцию Create folder reference for any added folders.
7. Запускаем, работает.

С кокос-x либой действуем один в один, там правда из ресурсов не нужно картинки для кнопок дропать.
